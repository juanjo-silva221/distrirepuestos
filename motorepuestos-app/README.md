# MotoRepuestos Frontend

Frontend en React/Vite para el proyecto MotoRepuestos.

## Desarrollo

```bash
npm install
npm run dev
```

## API

El frontend consume la API en `http://localhost:3000/api` por defecto. Si necesitas cambiarla, define `VITE_API_BASE_URL`.

## Nota

La documentación completa de backend, Docker y Postman está en el `README.md` de la raiz del proyecto.
